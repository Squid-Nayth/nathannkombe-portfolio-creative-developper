Application de gestion des employés des ligues – M2L

application console pour gérer des ligues et employés, avec persistance configurable, tests unitaires et architecture orientée objet.

Résumé du projet

Ce projet Java implémente un gestionnaire de personnel orienté ligues, conçu pour illustrer des bonnes pratiques de conception orientée objet et la persistance des données. L'application fournit un modèle GestionPersonnel (singleton) gérant des Ligue et des Employe, avec deux modes de persistance interchangeables : sérialisation et accès JDBC. Le code inclut une couche de passerelle pour isoler la stratégie de sauvegarde, des tests unitaires et une interface console pour les opérations courantes (création de ligues, gestion des droits, sauvegarde/restauration).

Cette application de gestion du personnel démontre ma capacité à modéliser un domaine métier, isoler les stratégies de persistance et fournir une suite de tests unitaires pour assurer le bon fonctionnement du programme.

Ce projet a été réalisé en équipe avec Java SE, avec l’IDE Eclipse, avec MySQL, avec Git et Github, avec ce projet j’ai pu appliquer mes connaissances en programmation orienté objet sur un projet concret
